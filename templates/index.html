Welcome to Python Football Manager

Python Football Manager is an online game where you can build a team, play matches and win cups. You have the opportunity to use a prebuild teams and try a quick game or spend some time by joining one of the cups available here.

But perhaps the most interesting part of PFM is the multiplayer game option. You can enjoy this game with a friend and see who is the best football manager.

The game is based on Python 3.10 and Flask.
